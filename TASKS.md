- [ ] **Демо-гейт на собственном домене.** На `crm.oceanglass.kg` голый
  вход упирается в гейт «Доступ к демо по ссылке» (нужен `?demo=…`/`?owner=…`).
  Раз это собственный домен клиента — пропускать гейт на этом хосте.
  → `initGate()` / `gateStatus()` в `data.js`.
- [x] **Дубликат имени сотрудника.** `u_pm` (менеджер) и `u_ps` (замерщик)
  оба «Платон Цай». → замерщик переименован в «Данияр Оспанов» в `buildSeed()`.
- [ ] **Расчёт опций по длине.** В `constrPrice` опции с `per:'м'` считаются
  как `2×ширина` (пометка `// periметр-ish`). Уточнить: подоконник/отлив — по
  ширине, откосы — по периметру проёма. → `constrPrice` в `data.js`.
