## Баги / нестыковки

- [ ] **Демо-гейт на собственном домене.** На `crm.oceanglass.kg` голый
  вход упирается в гейт «Доступ к демо по ссылке» (нужен `?demo=…`/`?owner=…`).
  Раз это собственный домен клиента — пропускать гейт на этом хосте.
  → `initGate()` / `gateStatus()` в `data.js`.
- [x] **Дубликат имени сотрудника.** `u_pm` (менеджер) и `u_ps` (замерщик)
  оба «Платон Цай». → замерщик переименован в «Данияр Оспанов» в `buildSeed()`.
- [x] **Расчёт опций по длине.** Подоконник/отлив считаются по ширине,
  откосы — по периметру проёма (2 высоты + верх). Вынесено в `extraLength()`,
  откосы переведены на `per:'периметр'`. → `constrPrice`/`EXTRAS` в `data.js`.

## Функциональность

- [x] **Рабочий поиск в топбаре.** Живой поиск по клиентам и сделкам
  (имя/телефон/адрес/комментарий), выпадающий список, клик открывает карточку,
  результаты гейтятся правами роли. → `globalSearch()` в `main.js`, разметка
  в `renderShell()` (`ui.js`), стили `.search-dd` в `styles.css`.
- [x] **Уведомления (колокольчик).** Модалка с реальной лентой последних событий
  из `DB.activity` (иконка по типу события, кто и когда). → `notifModal()` в
  `ui.js`, action `notif` в `main.js`.
- [x] **Экспорт/печать КП.** Кнопка «Печать / PDF» в модалке КП открывает чистый
  документ и вызывает печать (можно сохранить в PDF). Разметка вынесена в
  `kpDocHtml()`, функция печати — `printKp()`. → `modules2.js`, action в `main.js`.
